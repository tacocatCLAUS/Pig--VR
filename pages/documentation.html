About Pig VR

As an apple user it might seem that cheap VR is not developed for you. When I got a google cardboard I looked up some programs for it and every link I clicked on sent me to the play store. The Development Team at Pig VR made it so that there is a program that allows for a nice VR experience on your iphone.

How It Works

Pig Vr is coded in html/css/js the classic web development trio. Pig Vr does take advantage of a html plugin called A-Frame. A-Frame allows for an easy way to develop for a webVR experience. For the interactivity element of the program we used a combination of A-Frame's built in interactivity and vanilla javascript.

Downloads

Pig VR is constantly being updated by the Pig VR team. In every version section we talk about the improvements, updates, and flaws of the version. Some versions are need more computing power than others or need you to click out of something out of VR. If you want to try the absolute newest version even if it has bugs you can download the Experimental Version.
